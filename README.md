# Learning Micro-Academy

A learning platform for bite-sized courses and continuous learning. Explore our comprehensive course library.

## Overview

Learning Micro-Academy is a modern learning platform designed to help learners engage with courses on topics like:

- User Experience (UX) Design
- Design Systems
- Web Development
- Product Strategy
- Data Analytics
- Energy Efficiency
- Sustainability Practices
- And many more learning topics

## Features

- **Course Library**: Browse and explore a comprehensive collection of courses with detailed lessons
- **Course Selection**: Browse and select from a variety of learning topics
- **Featured Courses**: Discover highlighted courses each week
- **Learning Assistant**: AI-powered recommendations for personalized learning paths
- **Progress Tracking**: Monitor Your Active Courses and completed courses
- **Mobile Ready**: Built with Capacitor for iOS and Android deployment

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Mobile Development

For iOS:

```bash
npm run cap:ios
```

For Android:

```bash
npm run cap:android
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Capacitor (for mobile apps)
